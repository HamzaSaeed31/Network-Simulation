## 📖 About

A Computer Networks project designing and simulating a multi-router network topology using **Cisco Packet Tracer**. The simulation covers IP addressing, subnetting, routing protocols and connectivity verification across a LAN/WAN environment.

## ✨ Features

- 🗺️ Multi-router WAN topology design
- 🔢 IP addressing and subnetting scheme
- 🔄 Static and dynamic routing configuration
- ✅ End-to-end connectivity testing with ping/traceroute
- 📋 Detailed IP address table documentation

## 🛠️ Tech Stack

| | |
|---|---|
| Tool | Cisco Packet Tracer |
| Protocols | TCP/IP, OSPF / Static Routing |
| Concepts | Subnetting, VLANs, NAT |

## 🚀 Getting Started

### Prerequisites
- [Cisco Packet Tracer](https://www.netacad.com/courses/packet-tracer) (free with Cisco NetAcad account)

### Open the simulation

```
1. Clone this repo
2. Open Packet Tracer
3. File → Open → select projecttemp.pkt
4. Use the simulation mode to inspect traffic flow
```
